~Run the GUI.py file to get the Sudoku game to run.
One can insert the answers into the blanks in the gui and press ENTER to insert .
DELETE to delete an entered entry.
If the entered value is incorrect, amessage indicating ERROR will be displayed in for of a X mark down in the window.

The Backtracking aqlgorithm has been implemented to check and verify the result of the entry made into the blanks.
 
~Another seperate program "SimpleSudokusolve" can be used to diplay the working of backtracking algorithm in the program to find the solution for the program.
